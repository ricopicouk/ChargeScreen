# ChargeScreen

This is the public home of the ChargeScreen website, documentation, release
information, and compiled firmware downloads.

## Firmware source availability

This repository has not received firmware source-code updates for some time.
A decision has now been made to continue ChargeScreen firmware development as
closed-source software.

Future firmware source code will be maintained privately and will not be
published here. Only compiled firmware binaries will be made available
publicly.

Source code previously published in this repository remains available in its
existing Git history and remains subject to the licence terms included with
those historical versions.

## Website

The public website is hosted from the `docs` folder using GitHub Pages:

- [chargescreen.co.uk](https://chargescreen.co.uk/)

## Firmware downloads

Compiled firmware and release notes are published through GitHub Releases:

- [ChargeScreen releases](https://github.com/ricopicouk/ChargeScreen/releases)

No firmware source-code updates are planned for this public repository.
