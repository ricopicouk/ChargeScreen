Beta 3 release - beta3.0

This folder is a snapshot of the ChargeScreen firmware promoted to Beta 3.

Confirmed state:
- firmware version shows as beta3.0
- gradient drawing has been removed; battery and solar pages use solid backgrounds
- labels can be toggled on/off from settings page 2
- labels on/off uses identical value/status positions
- battery status always stays at the bottom
- solar status only appears at the bottom
- battery selector uses Victron or EcoW
- settings has a second swipe page with Battery and Labels controls
- firmware/settings info is available on the round screen by holding Demo mode for 3 seconds
- keys are entered and cleared from the WiFi settings page
- missing Victron keys show "Add key via WiFi" instead of decode error
- Eco-Worthy/JBD battery password storage is available on the WiFi settings page
- Victron solar BLE decrypt/parsing is retained
- solar page uses warm amber styling
- shunt demo values are 69%, 14.7 V, 30.0 A, 441 W, and 27.2 Ah
- demo mode is saved from the on-device settings screen
- screen rotation is saved from the on-device settings screen
- WiFi can be toggled on and off from settings
- VE.Direct and accelerometer rotation code are not included

Private settings from include/config.h are intentionally not included.
