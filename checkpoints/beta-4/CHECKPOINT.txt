Beta 4 release - beta4.0

This folder is a snapshot of the ChargeScreen firmware promoted from Beta 3.

Confirmed changes from Beta 3:
- temporary GPIO diagnostic page from the prior experiment has been removed
- BLE Capture is launched from the WiFi settings page, not from swipe navigation
- BLE Capture records raw BLE advertisements to a LittleFS CSV file for 5 minutes
- while capture is active, normal Victron/Solar/Eco-Worthy dashboard decoding is paused
- capture CSV is named after the strongest BLE device or address seen
- the round screen switches into a capture display with countdown, packet count, strongest device, RSSI, Stop & Save, and Cancel
- the capture display updates stable fields without constantly repainting the whole screen
- saved capture CSV can be downloaded from the existing ChargeScreen WiFi settings page
- screen dimming and screen-off timeout from the ChargeScreen checkpoint have been merged
- backlight brightness can be cycled between Max, Mid, and Low on settings page 2
- screen timeout can be changed on settings page 2 or precisely via the WiFi settings page
- the screen wakes on deliberate touch and ignores that wake touch until release
- firmware version is generated as beta4.01, beta4.02, etc. on sequential builds
- web firmware upload is available from the ChargeScreen WiFi settings page
- dual OTA app partitions are included for web firmware updates
- a LittleFS data partition is included for capture storage

Usage:
- Open Settings, start WiFi, connect to ChargeScreen, and visit http://192.168.4.1/.
- Read the BLE Capture explanation and press Start 5 Minute BLE Capture.
- The round screen changes to capture progress and offers Stop & Save or Cancel.
- Use Download BLE CSV when a capture is ready.
- Swipe to settings page 2 to adjust Timeout and Dimming.

Private settings from include/config.h are intentionally not included.
